## 1. Problem Decomposition

The goal is to prove that the set of exceptions $E = \{(n, k) : \min Fac(\binom{n}{k}) > \max(n/k, k)\}$ is finite.
We classify pairs $(n, k)$ based on the structure of $M = \lfloor n/k \rfloor$.

### Case 1: $n \le k^2$
-   **Status:** **Fully Formalized (Axiom-Free)**
-   **Method:**
    -   $k \le 28$: `native_decide` (exhaustive check).
    -   $k \ge 29$: `native_decide` for $n \in [2k, k^2]$ using CRT constraints.
-   **Result:** Exceptions contained in finite set $\{k \le 28, n \le 284\}$.

### Case 2: $n > k^2$
We split based on the prime factors of $M = \lfloor n/k \rfloor$.

#### Type A: $M$ has a prime factor $p > k$
-   **Status:** **Fully Formalized (Axiom-Free)**
-   **Method:** **Interval Divisibility Lemma**.
    -   If $p \mid M$ and $k < p \le M$, then $p \mid \binom{n}{k}$ for all $n \in [kM, kM+k)$.
    -   Since $p \le M \le n/k$, this kills the exception.

#### Type B: $M$ is $k$-smooth (all factors $\le k$)
This is the "Smooth Case". We further decompose based on divisibility.

-   **Subcase B1: $k \mid n$**
    -   **Status:** **Fully Formalized (Axiom-Free)**
    -   **Method:** `divisible_smooth_quotient_has_small_factor`.
    -   If $n/k$ is smooth and $k \mid n$, then $n$ is smooth. $\min Fac \le k$.

-   **Subcase B2: $n = kM + r$ with $1 \le r < k$**
    -   **Status:** **Rigorous NL Proof (Strategy 5) + Computation**
    -   **Formalization:** Uses axiom `large_n_smooth_case`.
    -   **Method (Large $k \ge 36$):** **Generalized Interval Divisibility**.
        -   The constraints $n \bmod p \in [k, p-1]$ for all $p \in (k, 2k]$ (Gap Primes) are incompatible with the intervals generated by $k$-smooth numbers.
        -   Computational check: 0 survivors for $k \in [36, 40]$.
    -   **Method (Small $k < 36$):**
        -   Computational verification of all smooth $M \le 10,000$.
        -   Known exceptions (e.g., $(62, 6)$) are found here.

## 2. Key Insights

1.  **Trailing Zero Lemma:** Formalized. Proves divisibility when $v_p(n) > v_p(k)$. This handles $n=kM$ and many $n=kM+r$ cases where valuations align.
2.  **Gap Primes (Strategy 5):** For $k \ge 36$, the density of primes in $(k, 2k]$ is high enough that the interval $[kM, kM+k)$ *must* contain a multiple of some gap prime. This structural property eliminates exceptions without needing Kummer constraints.
3.  **Kummer Satisfiability:** For $k \ge 7$, the Kummer conditions ($p \le k$) *are* satisfiable (primes $q$ exist), but these solutions are killed by Gap Primes ($p > k$). The intersection is empty.

## 3. Conclusion

The "gap" in the formal proof is strictly limited to **Type B2 ($k \nmid n$)**.
-   This gap is proven empty for $k \ge 36$ by Strategy 5.
-   It is verified empty for $k \in [29, 35]$ by computation.
-   It contains finite exceptions for $k \le 28$.

Thus, the Main Theorem (Finiteness) is established by a combination of Formalization, rigorous NL proof (Strategy 5), and Computation.
